# Basic HTML Layout/Style using Jade/Sass

Exemple site :
http://azgskull.github.io/basiclayout/

##Config's Steps:
####1.Gulp config

```
$ npm install gulp
```
```
$ npm install gulp-sass --save-dev
```
```
$ npm install gulp-jade --save-dev
```

And then Start watching
```
$ gulp watch
```

Enjoy !! :D
